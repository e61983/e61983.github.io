前言

最近發現家中的 NAS 速度 “好像” 沒有達到筆者的預期。所以想來實際測看看可以目前的網路環境可以跑到什麼程度。於似乎這一篇就誕生了。

主要內容

測試的 NAS 型號為: DS214。

在 Synology NAS 運行 iperf 服務

  • 安裝 iperf3
1
2
3
4

ssh synology-nas
sudo -i
synogear install
synogear list # 檢視 iperf
  • 執行
1

iperf3 -s
  • 輸出
1
2
3
4

(synogear) root@synology-nas:~# iperf3 -s
-----------------------------------------------------------
Server listening on 5201
-----------------------------------------------------------
